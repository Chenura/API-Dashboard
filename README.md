# API Testing Dashboard

A modern API Testing Dashboard inspired by Postman.

## Features

- REST API Testing
- GraphQL Support
- Authentication Tokens
- API Collections
- Request History
- Environment Variables
- Syntax Highlighting
- WebSocket Testing
- Import / Export Collections
- Multiple Tabs
- Dark Modern UI

## Technologies

- HTML
- CSS
- JavaScript
- Highlight.js
